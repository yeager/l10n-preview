# L10n Preview

Preview PO/TS translations in a simulated UI layout.

Built with GTK4/Adwaita. Part of the [Danne L10n Suite](https://github.com/yeager/debian-repo).

## Installation

### Debian/Ubuntu
```bash
sudo apt install l10n-preview
```

### Fedora/RPM
```bash
sudo dnf install l10n-preview
```
